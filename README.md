# Quick Technical SEO Analyzer 🔍

A lightweight, efficient Command Line Interface (CLI) tool designed for web developers, webmasters, and SEO specialists to perform rapid technical SEO audits on targeted web pages.

## Features ✨

- **Title Tag Audit:** Validates presence, text, and character length optimization.
- **Meta Description Check:** Inspects the description attribute for essential SEO presence.
- **Heading Hierarchy Analysis:** Audits `<h1>` tags to ensure correct semantic structure and warns about missing or multiple H1 tags.
- **Image Accessibility Audit:** Automatically scans all `<img>` tags for missing `alt` attributes to improve image SEO and accessibility.

## Installation 🚀

Clone the repository and install the required dependencies:

```bash
git clone [https://github.com/hamidjavadipoya/quick-seo-analyzer.git](https://github.com/hamidjavadipoya/quick-seo-analyzer.git)
cd quick-seo-analyzer
pip install beautifulsoup4 requests

```
## Usage 🛠️
Run the script from your terminal by providing the target website URL:
```bash
python seo_analyzer.py [https://example.com](https://example.com)

```
